# starter-lib

一个用于创建 npm 包的库模板。

[English Document](./README.md)

## 开发

- 安装依赖:

```bash
pnpm install
```

- 运行单元测试:

```bash
pnpm test
```

- 构建库:

```bash
pnpm build
```
