# starter-lib

A starter for creating a package.

[中文文档](./README.zh-CN.md)

## Development

- Install dependencies:

```bash
pnpm install
```

- Run the unit tests:

```bash
pnpm test
```

- Build the library:

```bash
pnpm build
```
